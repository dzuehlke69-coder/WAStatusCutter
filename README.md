# WAStatusCutter v2.0.0 🎬

Ein schlankes Android-Tool, um lange Videos in perfekte 59-Sekunden-Stücke für den WhatsApp-Status zu schneiden – ohne Qualitätsverlust durch Re-Encoding.

## Features
* **Schneller Schnitt:** Nutzt MediaMuxer für blitzschnelles Splitten ohne Umrechnung.
* **Smart Audio:** Behält die Audiospur bei (kleiner "Hacker" am Start technisch bedingt).
* **Automatischer Upload:** Übergibt die Schnipsel direkt an WhatsApp.
* **Auto-Reverse:** Sendet Dateien in umgekehrter Reihenfolge, damit sie im Status chronologisch korrekt erscheinen (1, 2, 3...).
* **Clean Storage:** Löscht alte Schnipsel automatisch vor jedem neuen Vorgang.
* **BTC Support:** Integrierter Spenden-Dialog mit QR-Code.

## Bedienung
1. App starten.
2. Auf **Video wählen** tippen.
3. Warten, bis der Schnitt fertig ist.
4. In WhatsApp **Mein Status** wählen und absenden.
